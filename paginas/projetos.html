Pequenos projetos desenvolvidos

Sempre tive vontade de aprender sobre o universo da tecnologia da informação, mas acabei seguindo um caminho um pouco diferente que também me deu oportunidades de aprender muitas coisas, mas recentemente decidi aprender coisas novas e entrar neste universo que está cada dia maior, e poder contribuir com esta história.

Seguem abaixo alguns pequenos projetos feitos para auxiliar meu aprendizado:

Campo minado

Um jogo simples feito feito para estudar algoritmos com a linguagem C.

Download do jogo compilado: Campo minado

Código fonte no github: https://github.com/timzejr2/CampoMinado.git

Galaxy Shooter

Um jogo feito durante o curso Guia Definitivo para Desenvolvimento de Jogos com Unity utilizando o motor gráfico Unity, Visual Studio e a linguagem C#

Acessar o jogo online em webGL: Galaxy Shooter

Observação: Pode ser necessário habilitar webGL no navegador

Para configurar webGL no Google Chrome digite "chrome://flags" na barra de endereços e ative a opção "enable webGL"

Maquete interativa no navegador

Modelo de maquete feita com Blender e com o motor gráfico Unity e compilado no formato webGL, que permite a maquete ser acessada pelo navegador.

Acessar a maquete online em webGL: Maquete interativa

Observação: Pode ser necessário habilitar webGL no navegador

Para configurar webGL no Google Chrome digite "chrome://flags" na barra de endereços e ative a opção "enable webGL"

FausteraGame

Jogo feito usando os recursos da engine Unity, Visual Studio, adobe Fuse, Mixamo e a linguagem C#, inspirado em um quadro clássico da televisão.

Acessar o jogo online em webGL: Bridgefall

Observação: Pode ser necessário habilitar webGL no navegador

Para configurar webGL no Google Chrome digite "chrome://flags" na barra de endereços e ative a opção "enable webGL"

Painel de Fastfood

Uma página feita durante as aulas eletivas de desenvolvimento Web, usando html, CSS, JavaScript e o banco de dados não relacional Firebase

Código fonte no github: https://github.com/timzejr2/Fastfood.git

Plataforma para estudo

Página para auxiliar a organização de determinados estudos através das metodologias Pomodoro e Carnell feita durante as aulas de engenharia de software em grupo utilizando a metodologia Scrum.

Código fonte no github: https://github.com/grupo5es/Teste1Grupo.git

Acesso a página: https://github.com/grupo5es/Teste1Grupo.git

Calculadora React

Uma calculadora feita usando a biblioteca React durante um curso de desenvolvimento web da Cod3r na Udemy

Código fonte no github: https://github.com/timzejr2/calculadora

Chatbot

Modelo de chatbot para atendimento de uma pizzaria feito no IBM Watson.

Modelo funcional do chatbot: Chatbot pizzaria
