Pequenos projetos desenvolvidos

Sempre tive vontade de aprender sobre o universo da tecnologia da informação, mas acabei seguindo um caminho um pouco diferente que também me deu oportunidades de aprender muitas coisas, mas recentemente decidi aprender coisas novas e entrar neste universo que está cada dia maior, e poder contribuir com esta história.

Seguem abaixo alguns pequenos projetos feitos para auxiliar meu aprendizado:

Campo minado

Um jogo simples feito feito para estudar algoritmos com a linguagem C.

Download do jogo compilado: Campo minado

Código fonte no github: https://github.com/timzejr2/CampoMinado.git

Galaxy Shooter

Um jogo feito durante o curso Guia Definitivo para Desenvolvimento de Jogos com Unity utilizando o motor gráfico Unity, Visual Studio e a linguagem C#

Acessar o jogo online em webGL: Campo minado

Observação: Pode ser necessário habilitar webGL no navegador

Para configurar webGL no Google Chrome digite "chrome://flags" na barra de endereços e ative a opção "enable webGL"

Bridgefall

Jogo feito usando os recursos da engine Unity, Visual Studio, adobe Fuse, Mixamo e a linguagem C#

Acessar o jogo online em webGL: Campo minado

Observação: Pode ser necessário habilitar webGL no navegador

Para configurar webGL no Google Chrome digite "chrome://flags" na barra de endereços e ative a opção "enable webGL"

Painél de Fastfood

Uma página feita durante as aulas eletivas de desenvolvimento Web, usando html, CSS, JavaScript e o banco de dados não relacional Firebase

Código fonte no github: https://github.com/timzejr2/CampoMinado.git

Calculado Reat

Uma calculadora feita usando a biblioteca React

Código fonte no github: https://github.com/timzejr2/CampoMinado.git
